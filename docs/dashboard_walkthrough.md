# Dashboard Walkthrough

## Revenue Tab

Use this tab to understand overall sales performance.

- KPI cards show revenue, orders, customers, and average order value.
- Monthly trend shows revenue movement over time.
- Country chart shows geographic concentration.
- Heatmap shows demand patterns by day and hour.

## Products Tab

Use this tab to identify products that deserve attention.

- Top products by revenue help merchandising and inventory decisions.
- Units sold help separate high-volume products from high-price products.

## Customers Tab

Use this tab to explain RFM-style thinking.

- Orders represent frequency.
- Revenue represents monetary value.
- Recency days show how recently a customer purchased.

## SQL Insights Tab

Use this tab to show that the dashboard is backed by repeatable SQL outputs, not manual spreadsheet work.
